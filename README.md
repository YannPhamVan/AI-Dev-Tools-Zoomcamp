# AI Dev Tools Zoomcamp

This repository showcases my work for the [AI Dev Tools Zoomcamp](https://datatalks.club/blog/ai-dev-tools-zoomcamp-2025-free-course-to-master-coding-assistants-agents-and-automation.html) organized by [DataTalks.Club](https://datatalks.club/).

---

## 1. Course overview

The **AI Dev Tools Zoomcamp** focuses on applying AI coding assistants, agents, and automation tools in real-world software development.  
It covers practical workflows for writing, testing, and maintaining code efficiently using AI, connecting AI agents to APIs, automating tasks, and leveraging CI/CD pipelines.

The course is free and open-source, taught by [Alexey Grigorev](https://www.linkedin.com/in/agrigorev/) and other experienced practitioners.

---

## 2. Prerequisites

- Basic programming skills (Python, JavaScript, or similar)  
- Familiarity with Git and the command line  
- No prior experience with AI-assisted coding tools required  

---

## 3. Tools and technologies covered

This course introduces a variety of AI-assisted and automation technologies, including:

- **GitHub Copilot**, **Cursor**, **Claude**, **OpenAI GPT models**  
- **FastAPI**, **React/TypeScript**, and **Django**  
- AI-powered agents and workflow automation (n8n)  
- CI/CD pipelines, testing frameworks, and API integrations  

---

## 4. Syllabus

The course is organized into six modules:

| Module | Title |
|--------|-------|
| 1 | Introduction to AI Dev Tools & Coding Assistants |
| 2 | End-to-End AI-Assisted Project (React + FastAPI) |
| 3 | Model Context Protocol (MCP) |
| 4 | Building an AI Coding Agent |
| 5 | AI for Testing, CI/CD & DevOps |
| 6 | Low-Code / No-Code Automation with n8n |
